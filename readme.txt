== Description ==

This plugin allows you to display a fully responsive grid of posts on your page.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/evermade-posts-plugin` directory
1. Activate the plugin through the 'Plugins' screen in WordPress
